JavaScript Basics

<script> Tag

In HTML, JavaScript code is incorporated between <script> and </script> tags.

Example

<script> document.getElementById("demo").innerHTML = "Hello JavaScript"; </script>

Historically, JavaScript may have used a type attribute: <script type="text/javascript">. The type attribute is not a requirement as JavaScript is the default scripting language in HTML.

JavaScript Functions & Events

A JavaScript function is a piece of JavaScript code that is executed when it's "called" or invoked.

For instance, a function can be invoked when a user-action event occurs, like clicking a button.

JavaScript in <head> or <body> Tags

You can place unlimited scripts in an HTML document.

Scripts can be located in the <body>, or in the <head> section of an HTML page.

JavaScript in <head> Tag

In the following example, JavaScript function is placed in the <head> section of an HTML page. The function is invoked when a button is clicked:


Example

<!DOCTYPE html> <html> <head> <script> function myFunction() {   document.getElementById("demo").innerHTML = "Text has been changed"; } </script> </head> <body> <h2>Demo: JavaScript in Head</h2> <p id="demo">A Sample Paragraph</p> <button type="button" onclick="myFunction()">Click Here</button> </body> </html>

JavaScript in <body> Tag

In the following example, JavaScript function is included in the <body> section of an HTML page. The function is called when a button is pressed:

Example

<!DOCTYPE html> <html> <body> <h2>Demo: JavaScript in Body</h2> <p id="demo">A Sample Paragraph</p> <button type="button" onclick="myFunction()">Click Here</button> <script> function myFunction() {   document.getElementById("demo").innerHTML = "Text has been changed"; } </script> </body> </html>

JavaScript Files

Scripts can be placed in separate external files which is practical when the same code is used across many different web pages.

JavaScript files have the .js extension.

Example

<script src="sample.js"></script>

Advantages of External JavaScript

Storing scripts in external files has numerous advantages:

  • Makes HTML and JavaScript easier to read and maintain
  • Separates HTML and code
  • Cached JavaScript files can improve page loads

Several scripts can be added to a single page by using several script tags:

Example

<script src="script1.js"></script> <script src="script2.js"></script>
