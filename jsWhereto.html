Locations for JavaScript Code

Using the <script> Tag

JavaScript code should go between <script> and </script> tags in an HTML page.

Example:

<script>
document.getElementById("example").innerHTML = "First JavaScript Line";
</script>

Remember that some older JavaScript examples might use a type attribute: <script type="text/javascript">. However, this attribute is not really necessary because JavaScript is the default scripting language in HTML.

JavaScript Functions and Triggers

A JavaScript function is a segment of code that is executed when it is "called" or "invoked".

A function might be invoked during an event, such as when a user clicks a button.

You'll learn more about functions and events in upcoming lessons.

Placing JavaScript in the <head> or <body> of Your HTML Page

You can insert any number of scripts into an HTML document. These scripts can be placed in the <body>, in the <head> section, or in both.

JavaScript in <head>

For example, a JavaScript function can be placed inside the <head> section. This function might be invoked when a button is clicked:

Example

<head>
<script>
function ourFunction() {
document.getElementById("example").innerHTML = "Text Changed.";
}
</script>
</head>

JavaScript in <body>

In another example, a JavaScript function can be placed within the <body> section and this might be triggered when a button is clicked:

Example

<body>
<script>
function ourFunction() {
document.getElementById("example").innerHTML = "Text Changed Again.";
}
</script>
</body>

Putting scripts at the bottom of the <body> element can boost the display speed because script interpretation can decelerate the display.

Using External JavaScript Files

Scripts can also be put in separate, external files. An external JavaScript file may be very useful when the same code is used in various web pages. JavaScript files have the .js file extension.

An external script can be used by placing the script file's name in the src (source) attribute of a <script> tag. This script tag can be located in the <head> or <body> as required. The script operates as if it is positioned exactly where the <script> tag is. Remember, external scripts cannot contain <script> tags.

Benefit of Using External JavaScript Files

Keeping scripts in different, external files has multiple advantages:

  • Separates HTML and coding
  • Makes HTML and JavaScript easier to read and maintain
  • JavaScript files kept in cache can speed up page loading

If you want to add multiple script files to a single page, simply use multiple script tags:

Example

<script src="script1.js"></script>
<script src="script2.js"></script>

External Script References

An external script can be linked in 3 different ways:

  • With a complete URL
  • With a file path (like /javascript/)
  • Without a path
