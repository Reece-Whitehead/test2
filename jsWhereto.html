JavaScript Positioning

The <script> tag is used in HTML to insert JavaScript code. This tag can be found between <script> and </script>.

A simple example:

<script>
document.getElementById("demo").innerHTML = "My First JavaScript";
</script>

Old JavaScript versions may apply the type attribute: <script type="text/javascript">. But, this is no longer required. In HTML, JavaScript is the default scripting language.

Functions and Events in JavaScript

A function in JavaScript refers to a defined block of code which gets executed when called. An event, such as a click action, can trigger these functions.

JavaScript in HTML

You can insert multiple scripts in an HTML document. These scripts could be positioned in the <body>, or in the <head> section, or both.

Example of JavaScript in <head>

In this example, a function is defined in the <head> section and is triggered when a button is clicked:

<!DOCTYPE html>
<html>
<head>

}
</script>

External JavaScript

JavaScript can also be stored in external files. Such an approach is advantageous when the same JavaScript code is utilized across multiple web pages. These files are saved with extension .js.

Here is an example of an external file named myScript.js:

function myFunction() {
  document.getElementById("demo").innerHTML = "Text changed.";
}

To add the external script, use the src attribute in the <script> tag. The script behaves as though it was placed exactly where the <script> tag is located.

<script src="myScript.js"></script>

Placing External JavaScript

There are several benefits in storing scripts in external files:

     
  • Maintains a separation between HTML and code.
    •  
  • Makes both HTML and JavaScript clearer and easier to manage
    •  
  • Cached JavaScript files can quicken page loads

External Script Linking Methods

There are three ways an external script can be referenced:

     
  1. With a full URL (web address).
  2. With a file path.
    3.  
  3. Without a path.
Examples: 
<script src="https://www.example.com/js/myScript.js"></script>
<script src="/js/myScript.js"></script>
<script src="myScript.js"></script>
