Getting Started with JavaScript

This overview illustrates the capabilities of JavaScript.

JavaScript and HTML Interactions

JavaScript can modify HTML content using various methods, like getElementById(). For instance, it can locate an HTML element by its id ("demo") and alter its content (innerHTML) to display "Hello JavaScript":

document.getElementById("demo").innerHTML = "Hello JavaScript";

Note that JavaScript supports both double and single quotes:

document.getElementById('demo').innerHTML = 'Hello JavaScript';

JavaScript Affects HTML Styles (CSS)

JavaScript can transform the style properties of an HTML element. This basically alters an HTML attribute:

document.getElementById("demo").style.fontSize = "35px";

JavaScript Can Manipulate Visibility of HTML Elements

JavaScript can hide HTML elements by changing the display style:

document.getElementById("demo").style.display = "none";

Similarly, JavaScript can make hidden HTML elements visible by adjusting the display style:

document.getElementById("demo").style.display = "block";

Interesting Facts

Remember, JavaScript and Java are entirely separate languages, with distinct concepts and designs. Brendan Eich invented JavaScript in 1995, and it became an ECMA standard in 1997. The standard is officially named ECMA-262, while the language is formally known as ECMAScript.
