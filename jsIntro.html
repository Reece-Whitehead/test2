Introduction to JavaScript

Following are some of the functionalities JavaScript can offer.

JavaScript Modularities: Changing HTML Content

One of the many tools in the JavaScript toolkit is the getElementById() method.

The snippet below demonstrates changing the content of an HTML element (identified by id="demo") to the phrase "Hello JavaScript":

Demonstration

document.getElementById("demo").innerHTML = "Hello JavaScript";

JavaScript allows both double and single quotes. Note the example:

Demonstration

document.getElementById('demo').innerHTML = 'Hello JavaScript';

JavaScript's Ability to Modify HTML Styles (CSS)

Changing the CSS of an HTML element is a variant of adjusting an HTML attribute. See the example:

Demonstration

document.getElementById("demo").style.fontSize = "35px";

Hiding and Showing HTML Elements with JavaScript

Hiding elements can be achieved by manipulating the display style attribute:

Demonstration

document.getElementById("demo").style.display = "none";

Displaying hidden HTML elements is achieved through a similar process:

Demonstration

document.getElementById("demo").style.display = "block";

Fascinating Facts

Though they share similar names, JavaScript and Java are entirely different in both their conceptual designs and language frameworks. JavaScript was created by Brendan Eich in 1995 and was standardized by the ECMA in 1997. ECMA-262 is the official name of this standard, while ECMAScript is the formal name given to the language.
