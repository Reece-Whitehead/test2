Getting Started with JavaScript

Below are some basic capabilities of JavaScript:

Modifying HTML Content with JavaScript

One commonly used JavaScript method for modifying HTML content is getElementById().

To illustrate, consider an HTML element with id="demo". JavaScript can alter the content of this element as follows:

Example

document.getElementById("demo").innerHTML = "Hello JavaScript";

Note that both single and double quotes can be used in JavaScript:

Example

document.getElementById('demo').innerHTML = 'Hello JavaScript';

JavaScript and HTML Styles (CSS)

JavaScript can also be used to change the style of HTML elements:

Example

document.getElementById("demo").style.fontSize = "35px";

Hiding HTML Elements with JavaScript

To hide an HTML element, you can modify the display style:

Example

document.getElementById("demo").style.display = "none";

Displaying Hidden HTML Elements with JavaScript

Correspondingly, JavaScript can show hidden elements by modifying the display style too:

Example

document.getElementById("demo").style.display = "block";

Interesting Facts

JavaScript and Java are totally distinct languages in both concept and design. JavaScript was created by Brendan Eich in 1995 and became an ECMA standard in 1997. The official name of the standard is ECMA-262, while the language is officially named ECMAScript.
