The Importance of JavaScript

JavaScript is one of the three essential languages web developers must master:

   1. HTML - to structure web page content

   2. CSS - to style and layout web pages

   3. JavaScript - to control web page behavior

You will be learning about all JavaScript versions:

  • Original Versions: ES1 - ES3 (Released between 1997-1999)
  • First Major Update: ES5 (2009)
  • Second Major Update: ES6 (2015)
  • Annual Updates (from 2016 to today)

Frequently Asked Questions
